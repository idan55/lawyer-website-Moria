# lawyer-website-Moria

Full-stack lawyer appointment booking platform.

## Stack
- Frontend: React + Vite + React Router + Tailwind + i18next
- Backend: Node.js (Express), Google Calendar API, PostgreSQL

## Run
1. Install dependencies:
   - `npm install`
   - `cd server && npm install`
2. Configure environment variables for frontend/backend.
3. Start backend:
   - `cd server && npm run dev`
4. Start frontend:
   - `npm run dev`
